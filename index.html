Ghost Blog Theme

A custom Ghost blog theme build on top of Ghostwriter. The theme is a simple AJAX driven theme released under the MIT License. It features clean typography, responsive style, SASS files etc.

Alfresco Login Reset

Alfresco Share new login page and an Activiti reset password workflow feature. The projects is a more secure version of the standard reset password feature in share extras. The login page is built with Material Design by Google.

Alfresco Admin Console

Alfresco 4.2x example of custom admin console page built with Aikau framework. The project is built with Maven 1.1.1 and shows how to override and extend the Share admin console to inject new menu values, pages, and data.
